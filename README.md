# TransVAR
### Tools for multi-wavelength follow-up and characterization of radio transient and variable candidates.

**TransVAR** is an open-source Python toolkit for the analysis, validation, and characterization of radio transient and variable source candidates. It provides reproducible workflows for candidate inspection, multi-wavelength cross-matching, variability analysis, photometric and radio light-curve analysis, dynamic spectra, period searches, luminosity calculations, and publication-quality figure generation.

Although originally developed for MeerKAT transient candidates produced by the TRON pipeline, TransVAR is designed to be generally applicable to radio transient and variable source studies.
